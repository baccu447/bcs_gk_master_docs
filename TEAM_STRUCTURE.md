## 👨‍💻 Team Members (Sub-Agents)

### 1. Laravel Dev (Backend Expert)
- **Primary Task:** Develop Laravel 11 API, Admin Panel, and AI Content Pipeline.
- **Focus:** Database migrations, Sanctum Auth, API Endpoints, and Gemini AI integration for question generation.
- **Runtime:** `subagent` or `acp`.

### 2. Flutter Dev (Mobile Expert)
- **Primary Task:** Build the Flutter Android/iOS application.
- **Focus:** UI/UX implementation, GetX/Riverpod state management, Hive offline-first sync, and AdMob integration.
- **Runtime:** `subagent` or `acp`.

### 3. QA Bot (Testing & Quality)
- **Primary Task:** Test APIs and Flutter widgets.
- **Focus:** Unit testing, feature validation, and bug reporting before Git push.
- **Runtime:** `subagent`.
